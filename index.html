Virtual Keyboard

This project is a fully responsive virtual keyboard built using HTML, CSS, and JavaScript. The keyboard is designed to replicate the functionality of a physical keyboard, allowing users to input text without the need for physical keys.
Key Features:
Responsive Design: The keyboard adjusts seamlessly across different screen sizes, ensuring a consistent user experience on desktops, tablets, and mobile devices.
Customizable Layout: Users can switch between different keyboard layouts, including QWERTY, AZERTY, and Dvorak.
Interactive Feedback: Keys highlight dynamically when pressed, providing visual feedback similar to a physical keyboard.
Multi-language Support: The keyboard can be easily adapted to support multiple languages and character sets.
Dark/Light Mode: Includes a toggle for dark and light mode, enhancing usability in various lighting conditions.
This project demonstrates my ability to create interactive, user-friendly interfaces and highlights my proficiency in front-end development, including responsive design, JavaScript functionality, and user experience optimization.
